── projects §03

dere private

persona-layered claude code wrapper with a daemon for sessions, emotions, and a knowledge graph. plugins bolt on workflow-specific behavior (code, taskwarrior, vault). typescript · bun · postgres · falkordb · hono · trpc

choir

local agent orchestrator in moonbit. expensive model plans (claude as team lead), cheap models implement in parallel git worktrees and file PRs upward. scaffold → fork → converge. moonbit · git · gh · zellij

moonsorter

browser ranking via the plackett-luce model. full rankings + pairwise comparisons, MM fitting, bootstrap confidence intervals, k-means tier clustering. moonbit compiled to wasm. moonbit · wasm · astro

verbum-diei

daily lectionary site. pulls vatican news rss, renders the reading from a local douay-rheims, glues in liturgical calendar (romcal), hours of prayer (breviarium), and optional llm marginalia. static, regenerated nightly. typescript · purescript · bun

moonbit ecosystem

moonbit libraries pulled out of choir / moonsorter as standalone tools: moontrace (structured tracing with otlp export), tempo (utc datetime + rfc 3339), pippa (terminal ui framework, bubbletea-style model-update-view). moonbit · wasm
